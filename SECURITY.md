# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

If you discover a security vulnerability in depfresh, please report it responsibly.

**Email:** [moises.nascimento@helios.sc](mailto:moises.nascimento@helios.sc)

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge your report within **3 business days** and aim to release a fix within **14 days** for confirmed issues.

Please **do not** open a public GitHub issue for security vulnerabilities.

## Scope

depfresh shells out to `uv` and `pip-audit`. Vulnerabilities in those tools should be reported to their respective maintainers:

- **uv:** https://github.com/astral-sh/uv/security
- **pip-audit:** https://github.com/pypa/pip-audit/security
